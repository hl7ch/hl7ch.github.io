Extension: If an encounter is caused by an accident

Official URL: http://fhir.ch/ig/ch-core/StructureDefinition/ch-ext-accident Version: 2.1.0
Active as of 2020-12-14 Computable Name: Accident

Copyright/Legal: CC-BY-SA-4.0

Extension to define an encounter that is caused by an accident and at which time the accident happened

Context of Use

This extension may be used on the following element(s):

  • Encounter.hospitalization

Usage info

Usage:

  • Use this Extension: CH Core Encounter
  • Examples for this Extension: Encounter/EncounterAccidentBroennimann
