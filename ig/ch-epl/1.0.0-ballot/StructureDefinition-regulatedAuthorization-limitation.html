This structure is derived from Extension

Summary

Complex Extension: A limitation to an authorisation for reimbursement of a drug, for a certain indication.

  • status: CodeableConcept: The status of the limitation
  • statusDate: date: The date at which it limitation status was set
  • period: Period: The period over which the reimbursement authorisation applies
  • reimbursementEndDate: date: The date at which it becomes no longer possible to be reimbursed
  • firstLimitationDate: date: The date at which it is first possible to be reimbursed
  • limitationIndication: Reference: Reference to a Clinical Use Definition resource that defines the limitation of the indication.
  • indicationCode: string: The indication code assigned by FOPH
