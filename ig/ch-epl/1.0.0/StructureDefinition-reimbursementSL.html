Documentation for this format

This structure is derived from Extension

Summary

Complex Extension: A set of information about the reimbursement of a drug

  • FOPHDossierNumber: Identifier: The dossier number of FOPH
  • status: CodeableConcept: The status of the reimbursement
  • statusDate: date: The date at which it reimbursement status was set
  • expiryDate: date: The date at which it reimbursement expires
  • listingStatus: CodeableConcept: The status of the listing
  • listingPeriod: Period: The period over which the reimbursement authorisation is listed
  • firstListingDate: date: The date at which it is first possible to be reimbursed
  • costShare: integer: The retention value
  • productType: CodeableConcept: The product type related to the reimbursement
  • gamme: CodeableConcept: The gamme related to the reimbursement
