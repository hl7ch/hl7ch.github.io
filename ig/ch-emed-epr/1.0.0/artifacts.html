Terminology: Value Sets

These define sets of codes used by systems conforming to this implementation guide.

Amount Quantity Unit Codes

Unit Codes for amount quantity units.

These are all unit codes about amount quantities, excluding time units.
Event Timings

Event Timings for ambulatory use.
Route of Administration

Valueset RouteOfAdministration filtered for ambulatory care use.

This value set is a subset of the mandatory value set (EDQM - RouteOfAdministration) that contains only values compatible with ambulatory care. We recommend implementers to focus on supporting this value set instead of the complete one.
Substance Administration Substitution Codes

Codes in use to allow or deny medication substitution.
Time Quantity Unit Codes

Unit Codes for time quantity units.

These are all unit codes about time quantities, excluding amount units.
