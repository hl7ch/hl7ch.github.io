ValueSet: Route of Administration (Experimental)

Official URL: http://fhir.ch/ig/ch-emed-epr/ValueSet/ch-emed-epr-route-of-administration Version: 1.0.0
Draft as of 2024-01-26 Computable Name: CHEMEDEPRRouteOfAdministration

Copyright/Legal: CC0-1.0

Valueset RouteOfAdministration filtered for ambulatory care use.

This value set is a subset of the mandatory value set (EDQM - RouteOfAdministration) that contains only values compatible with ambulatory care. We recommend implementers to focus on supporting this value set instead of the complete one.

References

This value set is not used here; it may be used elsewhere (e.g. specifications and/or implementations that use this content)
