Data Type Profile: Ratio (Amount / Time)

Official URL: http://fhir.ch/ig/ch-emed-epr/StructureDefinition/ch-emed-epr-ratio-amount-per-time Version: 2.0.0
Draft as of 2025-01-21 Computable Name: CHEMEDEPRRatioAmountPerTime

Copyright/Legal: CC0-1.0

A ratio of amount per time

Usage:

  • Use this DataType Constraint: Dosage (MedicationRequest), Dosage Split and Dosage (MedicationStatement / MedicationDispense)
