ValueSet: Amount Quantity Unit Codes (Experimental)

Official URL: http://fhir.ch/ig/ch-emed-epr/ValueSet/ch-emed-epr-amount-quantity-unit-code Version: 2.0.0
Draft as of 2025-01-21 Computable Name: CHEMEDEPRAmountQuantityUnitCode

Copyright/Legal: CC0-1.0

Unit Codes for amount quantity units.

These are all unit codes about amount quantities, excluding time units.
