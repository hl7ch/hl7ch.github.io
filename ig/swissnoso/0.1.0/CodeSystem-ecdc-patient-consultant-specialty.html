CodeSystem: ECDC Patient-Consultant Specialty

Official URL: http://fhir.ch/ig/swissnoso/CodeSystem/ecdc-patient-consultant-specialty Version: 0.1.0
Active as of 2022-04-05 Computable Name: ECDCPatientConsultantSpecialty

Copyright/Legal: CC-BY-SA-4.0

This code system contains specialty codes defined by the European Centre for Disease Prevention and Control (ECDC), used by Swissnoso in the context of monitoring and prevention of healthcare-associated infections. See original code system in European Centre for Disease Prevention and Control. Point prevalence survey of healthcare-associated infections and antimicrobial use in European acute care hospitals – protocol version 5.3. Stockholm: ECDC; 2016., page 40 f.

This Code system is referenced in the content logical definition of the following value sets:

  • SwissnosoSpecialty
