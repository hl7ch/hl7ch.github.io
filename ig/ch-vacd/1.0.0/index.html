Introduction

This implemenation guide is based on the CDA-CH-VACD - Immunization Content (eVACDOC) ART-DECOR® specification which has been published by eHealth Suisse. CDA-CH-VACD - Immunization Content (eVACDOC) defines the documents for the exchange of immunization and vaccination information. The documents are based on the IHE Technical Framework „Patient Care Coordination (PCC)“ with the profile „Immunization Content (IC)“. The IHE PCC himself references the HL7 Clinical Document Architecture (CDA).

The CH VACD implementation guide describes the FHIR representation of the defined documents for the exchange of vaccination and immunization information. This implementation guide is dependent on CH Core and CH EPR Term, which describe the Swiss specific context.

The following documents have been defined:

  • Immunization Administration document
  • Immunization Certificate document
  • Vaccination Record document
  • Immunization Recommendation Request document
  • Immunization Recommendation Response document
