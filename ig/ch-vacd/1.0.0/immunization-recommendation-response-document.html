Immunization Recommendation Response

  • FHIR document (Bundle)
  • Resource profiles

The Immunization Recommendation Response document describes the content and format of an Immunization Recommendation Response containing all immunization recommendations which can be made based on the data delivered by the immunization recommendation request. The response includes all data of the request. The patient data MUST be pseudonymized.

FHIR document (Bundle)

This exchange format is defined as a document type that corresponds to a Bundle as a FHIR resource. A Bundle has a list of entries. The first entry is the Composition, in which all contained entries are then referenced.
