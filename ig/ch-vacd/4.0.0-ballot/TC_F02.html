Test Case F 02: Add comment (note) to an existing AllergyIntolerance

Precondition

In the EPR of the patient there are several ImmunizationAdministration documents with AllergyIntolerance set.

Sequence

The HCP1 add a comment (note) to an existing AllergyIntolerance in an exitsting ImmunizationAdministration document. Therefor a new ImmunizationAdministration document will be created.

Result

The HCP1 in community C1 has created an ImmunizationAdministration document F_D2_HCP1_C1 which adds a comment to the AllergyIntolerance entry added in document F_D1_HCP1_C1.

In the resulting document only the AllergyIntolerance with the note entry will be shown.
