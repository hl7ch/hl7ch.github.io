Resource Profile: CH VACD Travel Information

Official URL: http://fhir.ch/ig/ch-vacd/StructureDefinition/ch-vacd-travel-information Version: 7.0.0-ballot
Active as of 2026-06-11 Computable Name: CHVACDTravelInformation

Copyright/Legal: CC0-1.0

Travel Information for ImmunizationRecommendation for travel vaccinations.

Scope and Usage

This Condition profile is used to define travel information to be sent with an Immunization Recommendation Request Message to a CDS System to receive a Immunization Recommendation Response Message with Immunization Recommendations in context to the travel destination.

Model

Usages:

  • Use this Profile: CH VACD Message Immunization Recommendation Request
  • Refer to this Profile: CH VACD Immunization Recommendation Request MessageHeader
  • Examples for this Profile: Condition/8-10-Condition

You can also check for usages in the FHIR IG Statistics
