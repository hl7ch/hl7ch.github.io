Test Case C 02: Correction of foreign entry

Precondition

The patient has an EPR. There is an ImmunizationAdministration document A_D3_HCP2_C2 created by HCP2 on the respository of the community C2.

Sequence

The HCP1 creates and ImmunizationAdministration document to correct an entry of HCP2 from communita C2 and stores it on the repository of the communit C1.

Result

The HCP1 in community C1 has created an ImmunizationAdministration document C_D2_HCP1_C1 which corrects some parameters of an Immunization entry of the document A_D3_HCP2_C2.

In the resulting document only the new/corrected immunization entry will be shown.
