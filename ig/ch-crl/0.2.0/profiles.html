Profiles defined as part of the CH CRL Implementation Guide

The following profiles set the minimum expectations to record health data associated with a Patient:

  • CH CRL Bundle Profile Definition of the bundle for reporting to the cancer registry
  • CH CRL Communication Profile Communication for the date of informing the patient about the cancer registration
  • CH CRL Composition Profile Definition of the composition for reporting to the cancer registry
  • CH CRL Condition Disease Profile Condition for diseases
  • CH CRL Condition Final Cause of Death Profile Condition for the final cause of death
  • CH CRL DocumentReference Profile Document reference for the report as pdf
  • CH CRL Observation Date of Incidence Profile Observation for the date of incidence
  • CH CRL Observation ICD-10 Profile Observation for the ICD-10 code
  • CH CRL Observation ICD-O-3 Behaviour Profile Observation for the ICD-O-3 behaviour code
  • CH CRL Observation ICD-O-3 Histological Grade Profile Observation for the ICD-O-3 histological grade code
  • CH CRL Observation ICD-O-3 Laterality Profile Observation for the ICD-O-3 laterality code
  • CH CRL Observation ICD-O-3 Morphology Profile Observation for the ICD-O-3 morphology code
  • CH CRL Observation ICD-O-3 Topography Profile Observation for the ICD-O-3 topography code
  • CH CRL Observation Lymphatic Invasion Profile Observation for the lymphatic invasion
  • CH CRL Observation Number of Examined Regional Lymph Nodes Profile Observation for the number of examined regional lymph nodes
  • CH CRL Observation Number of Involved Regional Lymph Nodes Profile Observation for the number of involved regional lymph nodes
  • CH CRL Observation Perineural Invasion Profile Observation for the perineural invasion
  • CH CRL Observation TNM Stage Group Profile Observation for the TNM stage group
  • CH CRL Observation Venous Invasion Profile Observation for the venous invasion
  • CH CRL Observation a-Prefix of pTNM Profile Observation for the a-Prefix of pTNM
  • CH CRL Observation cM Profile Observation for the TNM clinical distant metastases category
  • CH CRL Observation cN Profile Observation for the TNM clinical regional nodes category
  • CH CRL Observation cT Profile Observation for the TNM clinical primary tumor category
  • CH CRL Observation m-Suffix of pT Profile Observation for the m-Suffix of pT
  • CH CRL Observation pM Profile Observation for the TNM pathologic distant metastases category
  • CH CRL Observation pN Profile Observation for the TNM pathologic regional nodes category
  • CH CRL Observation pT Profile Observation for the TNM pathologic primary tumor category
  • CH CRL Observation y-Prefix of cTNM Profile Observation for the y-Prefix of cTNM
  • CH CRL Observation y-Prefix of pTNM Profile Observation for the y-Prefix of pTNM
  • CH CRL Organization Department Profile Definition of the organization department (hospital) for reporting to the cancer registry
  • CH CRL Organization Profile Organization of which the organization department forms a part
  • CH CRL Patient Profile Definition of the patient for reporting to the cancer registry
  • CH CRL Practitioner Profile Definition of the practitioner (resident physician) for reporting to the cancer registry
  • CH CRL Procedure Diagnostic Method Profile Procedure for the diagnostic method(s)
