CodeSystem: NKRS Variables

Summary

Defining URL:http://fhir.ch/ig/ch-crl/CodeSystem/nkrs-variables
Version:0.9.0
Name:NkrsVariables
Title:NKRS Variables
Status:Active as of 2022-02-04 02:14:21+0100
Definition:

NKRS codes for data elements for which there is no international code (e.g. LOINC or SNOMED CT) that accurately describes the meaning of the NKRS variable. This code system is based on 'NATIONAL CANCER DATA DICTIONARY, V 1.1, Part A, BASIC VARIABLES for Adults, Adolescents, and Children (23.12.2019)' (EN, DE, FR, IT) and 'NATIONAL CANCER DATA DICTIONARY, V 1.1, Part B1, SUPPLEMENTARY VARIABLES for Adults (15.10.2019)' (EN, DE, FR, IT).
