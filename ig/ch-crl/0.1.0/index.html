Swiss CRL (Cancer Registration Law) Implementation Guide

Introduction

The documentation for the hearing is available on the website of the National Institute for Cancer Epidemiology and Registration
(https://www.nicer.org > Krebsregistrierung > Partner im Gesundheitswesen).

Scope

  • Profiles - are useful constraints of core FHIR resources and datatype for Swiss use
  • Extensions - are FHIR extensions that are added for local use, covering needed Swiss concepts
  • Terminologies - are defined or referenced code systems and value sets for Swiss context
