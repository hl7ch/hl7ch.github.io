Title:NKRS - ICCC-3 Extended Code
Status:Active as of 2022-02-04 02:14:21+0100
Definition:

This data item records detailed division of the diagnostic group according to the third revision of the 1996 International Classification of Childhood Cancer (ICCC-3).

ICCC-3 code (Variable number: 3.9.3) in 'NATIONAL CANCER DATA DICTIONARY, V 1.1, Part A, BASIC VARIABLES for Adults, Adolescents, and Children (23.12.2019)' (EN, DE, FR, IT)
