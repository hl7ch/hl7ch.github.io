Extension: CH EMED Extension Dispense

Official URL: http://fhir.ch/ig/ch-emed/StructureDefinition/ch-emed-ext-dispense Version: 3.0.0
Active as of 2022-12-21 Computable Name: CHEMEDExtDispense

Copyright/Legal: CC-BY-SA-4.0

Extension to reference an external MedicationDispense and the Medication Dispense document that contains it

Context of Use

This extension may be used on the following element(s):

  • Element ID MedicationRequest
  • Element ID MedicationDispense
  • Element ID MedicationStatement
  • Element ID Observation

Usage info

Usage:

  • Use this Extension: CH EMED Observation (LIST), CH EMED Observation (PADV)
