Extension: CH EMED Extension Treatment Plan

Official URL: http://fhir.ch/ig/ch-emed/StructureDefinition/ch-emed-ext-treatmentplan Version: 6.0.0
Active as of 2025-12-16 Computable Name: CHEMEDExtTreatmentPlan

Copyright/Legal: CC0-1.0

Extension to reference an external MedicationStatement and the Medication Treatment Plan document that contains it

Context of Use

    This extension may be used on the following element(s)

    • Element ID: MedicationRequest
    • Element ID: MedicationDispense
    • Element ID: MedicationStatement
    • Element ID: Observation

Usage info

Usages:

  • Use this Extension: CH EMED MedicationAdministration (DIS), CH EMED MedicationDispense (DIS), CH EMED MedicationRequest (PRE), CH EMED MedicationStatement (CARD)... Show 2 more, CH EMED MedicationStatement (LIST) and CH EMED Observation (PADV)
  • Examples for this Extension: Bundle/1-2-MedicationDispense, Bundle/2-1-MedicationList, Bundle/2-2-PharmaceuticalAdvice, Bundle/2-4-MedicationDispense... Show 14 more, Bundle/2-6-MedicationPrescription, Bundle/2-7-MedicationCard-UUIDfullUrl, Bundle/2-7-MedicationCard, Bundle/PharmaceuticalAdvice-ChangeDosage, MedicationDispense/1-2-MedDispTriatec, MedicationDispense/2-1-MedDispTriatec, MedicationDispense/2-4-MedDispBeloczok, MedicationRequest/2-6-MedReqNorvasc, MedicationStatement/2-1-MedStatTriatec, MedicationStatement/2-7-MedStatBeloczok, MedicationStatement/2-7-MedStatNorvasc, MedicationStatement/NotRecommendedTimingCodes, Observation/2-2-ObsPharmaceuticalAdvice and Observation/Obs-ChangeDosage
