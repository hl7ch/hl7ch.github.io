Extension: CH EMED Extension Document Author

Extension to represent the author of the original document in the medication statement (for the Medication Card document)

The official URL for this extension is: 

http://fhir.ch/ig/ch-emed/StructureDefinition/ch-emed-ext-documentauthor

Status: active
Extension maintained by: HL7 Switzerland

Context of Use

This extension may be used on the following element(s):

  • MedicationStatement

Usage info

Usage:

  • Use this Extension: CH EMED MedicationStatement Profile (Medication Card document)
  • Examples for this Extension: Bundle/MedicationCard-DifferentAuthors
